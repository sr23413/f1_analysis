## Data Journalism Project
**What's Standing Between Kimi Antonelli and a World Championship**  
Analysis of Antonelli's 2025-2026 F1 performance data, comparing consistency, pace and teammate benchmarks across his debut season and early 2026.

## Data sources
- FastF1 — race and qualifying session data
- Jolpica — historical F1 results
- Google Trends — public search interest

## Tools
Python, Jupyter, pandas, seaborn, matplotlib, pytrends
